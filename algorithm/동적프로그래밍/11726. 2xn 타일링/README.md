## Solution

### 가짜 문제 정의

진짜 문제 : 2xn 타일링

가짜 문제 : 2xi 타일링

가짜 문제에 대해서 모든 배열을 채우고 나면 D[N]의 값은 2xn타일링의 값과 같게 되어 진짜 문제를 풀 수 있다.

### 초기값과 점화식

- Dp[1] = 1
- Dp[2] = 2

점화식의 경우, 타일의 길이가 가로 1, 세로 2이기 때문에 전체 타일의 늘어나는 길이는 최대 2가 된다. 따라서 Dp[i - 2]부터 살펴보면 된다.
