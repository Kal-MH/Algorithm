## Solution

### 풀고 싶은 가짜 문제 정의

진짜 문제를 먼저 써보고, 가짜 문제로 바꿔보자.

- 진짜 문제 : 주어진 N에 대해서 1, 2, 3의 합으로 표현하는 경우의 수
- 가짜 문제 : j에 대해서 1, 2, 3의 합으로 표현하는 경우의 수

### 가짜 문제를 풀면 진짜 문제를 풀 수 있는가

배열을 가득 채운다고 한다면 주어진 N에 대한 1, 2 3의 합으로 표현하는 경우의 수는 D[N]의 값과 같다.
