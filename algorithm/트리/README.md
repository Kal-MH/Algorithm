## 개념

### 트리란, 그래프의 특수한 형태

```java
특징 3가지

1. 모두 연결되어 있는 그래프 → 어떤 두 점을 골라도 간선을 타고 사이를 이동 가능
2. 사이클이 존재하지 않음
3. 정점 개수 |v| = 간선개수 |E|  + 1

다음 3가지 중 2가지를 만족하면 트리라고 할 수 있다.
```

### Rooted Tree

- Node
- Root
- Depth
    
    root노드로부터 도달하는 데 사용하는 간선의 개수(root는 깊이가 0이라고 할 수 있다.)
    
- Parent, Child, Ancestor, Sibling
- Leaf Node

### Tree의 키워드 예시

- 모든 정점사이에 잇는 경로가 존재하며 사이클이 존재하지 않는 경우만 고려
- 마을과 마을 사이 있는 N-1개의 길 존재, 모든 마을은 연결되어 있음

### 트리는 대부분 인접리스트를 이용해 구현한다.

트리의 경우, vertex 개수가 V라고 할 때, 간선의 개수는 E-1개가 됨으로 O(V^2)의 공간이 필요한 인접행렬의 경우, 공간낭비가 생길 수 있다.
