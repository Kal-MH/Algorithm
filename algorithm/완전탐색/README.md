## 개념

문제를 해결하기 위해 확인해야 하는 모든 경우를 전부 탐색하는 방법

```c
좋은 점
- 부분점수를 어기 좋다.

나쁜 점
- 전부 탐색하기에 시간 복잡도가 일반적으로 높다.
```

## 완전탐색의 종류

```c
N개 중
	- 1) 중복을 허용해서
	- 2) 중복 없이

M개를
	- A) 순서 있게 나열하기
	- B) 고르기
```
